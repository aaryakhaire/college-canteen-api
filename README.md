\# College Canteen API 🍽️



A Flask-based REST API for managing a college canteen system.



\## Features

\- Menu Management

\- Orders API

\- User/Admin endpoints (future scope)



\## Tech Stack

\- Python

\- Flask

\- PostgreSQL / SQLite (based on setup)



\## How to Run

1\. Clone the repo

2\. Create and activate virtual environment

3\. Install dependencies:

&nbsp;  pip install -r requirements.txt

4\. Run the server:

&nbsp;  python app.py



\## API Test

Use Postman / Insomnia to test endpoints.
